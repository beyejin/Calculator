# Java 계산기 프로젝트

## 📌 프로젝트 소개
Java를 활용하여 구현한 콘솔 기반 계산기 프로젝트입니다.  
사용자로부터 입력받은 두 수와 사칙연산 기호를 통해 계산을 수행하며,  
연산 결과를 저장하고 관리하는 기능을 포함하고 있습니다.

본 프로젝트는 단계별 요구사항을 기반으로 구현되었으며,  
객체지향 개념과 Java의 주요 문법을 학습하는 것을 목표로 합니다.

각 스텝별 구현 사항
## ⚙️ 각 스텝별 구현 사항

###Step 1 - 기본 계산기 기능
- 양의 정수 2개 입력받기
- 양의 정수가 아닐 경우 재입력 처리
- 정수 타입이 아닐 경우 예외 처리 후 재입력
- 사칙연산 기호 입력받기 (`+`, `-`, `*`, `/`)
- 올바르지 않은 연산자 입력 시 재입력
- 사칙연산 기호에 따라 계산 수행
- 나눗셈 시 분모가 0일 경우 숫자 입력 단계로 돌아가기
- 반복문을 이용하여 계산 지속 수행
- `exit` 입력 시 프로그램 종료
- 계산 반복 여부 입력받기

---

###Step 2 - 객체지향 & 컬렉션 활용
- 사칙연산 수행 후 결과를 반환하는 메서드 구현
- 연산 결과를 저장하는 컬렉션(`ArrayList`) 필드 생성
- 계산 기능을 별도의 `Calculator` 클래스로 분리
- 연산 결과를 컬렉션에 저장
- 컬렉션을 `private`으로 선언하여 캡슐화
- 가장 오래된 결과값을 삭제하는 기능 구현

---

###Step 3 - 확장 기능
- `Enum`을 활용하여 사칙연산 상수 집합 정의
- `Enum` 내부에 연산 수행 메서드 구현
- 제네릭을 활용하여 실수(Double) 입력 가능하도록 확장
- 람다 & 스트림(Stream)을 활용한 조회 기능 구현
  - 입력값보다 큰 결과값만 필터링하여 출력
